
# 인트로 화면 완전 재설계

## 현재 문제점
- 흰색 프로그레스 바가 중앙에 있고, 100% 후 별도 "로고 페이즈"로 전환됨
- 로고 페이즈에서 흰색 네모 박스가 나타나는 문제

## 레퍼런스 분석
1. **처음부터 브릿지 로고가 중앙에 위치**
2. **숫자가 올라가면서 로고가 점점 드러남** (가운데에서 좌우로 퍼지는 reveal)
3. **숫자는 로고 위에 겹쳐서 표시** (로고 내부 중앙)
4. **100% 완료 시 바로 페이드아웃** (별도 로고 페이즈 없음)

---

## 수정 계획

### PageIntro.tsx 전면 재작성

#### 구조 변경
- **phase 단순화**: `loading` → `expand` → `done` (logo 페이즈 제거)
- **흰색 프로그레스 바 제거**
- **브릿지 로고를 처음부터 표시**

#### UI 레이아웃
```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│        ┌───────────────┐            │
│        │  브릿지 로고   │            │
│        │    (reveal)   │            │
│        │               │            │
│        │    0 5 8      │  ← 숫자 겹침 │
│        │               │            │
│        └───────────────┘            │
│                                     │
│                                     │
└─────────────────────────────────────┘
```

#### 로고 Reveal 방식
- `clipPath: inset(0 ${50 - progress/2}% 0 ${50 - progress/2}%)`
- 0% → 로고 안 보임 (양쪽에서 50%씩 잘림)
- 50% → 로고 절반 보임
- 100% → 로고 전체 보임

#### 숫자 표시
- 로고 컨테이너 내부 중앙에 absolute로 배치
- 3자리 숫자 (`000` ~ `100`)
- 큰 폰트 사이즈 (`text-5xl sm:text-7xl`)
