
# 인트로 화면 프리미엄 업그레이드 계획

## 현재 문제점
- 로고 크기가 너무 작음 (w-20, h-20 = 80px)
- 배경 그라데이션 효과가 약함
- 전체적으로 임팩트 부족
- 애니메이션이 단순함

---

## 개선 방향

### 1. 로고 크기 및 스타일 대폭 강화
- 로고 크기: 80px → **160px (모바일) / 200px (데스크톱)**
- 로고 글로우 효과 추가 (drop-shadow)
- 브랜드명 폰트 사이즈 증가: 30px → **48px~56px**

### 2. 배경 효과 강화
- 중앙 글로우 크기 확대 및 강도 증가
- 다중 레이어 그라데이션 추가
- 미세한 노이즈/그레인 텍스처 오버레이

### 3. 애니메이션 개선
- 로고 스케일인 효과 (0.5 → 1)
- 글로우 펄스 강도 증가
- 페이드인 타이밍 최적화

---

## 수정 파일

| 파일 | 변경 내용 |
|------|----------|
| `src/components/PageIntro.tsx` | 로고 크기 확대, 배경 글로우 강화, 텍스트 크기 증가 |
| `src/index.css` | 인트로 애니메이션 keyframes 개선 |

---

## 변경 세부사항

### PageIntro.tsx 주요 변경
```
- 로고: w-20 h-20 → w-32 h-32 sm:w-40 sm:h-40
- 브랜드명: text-3xl → text-4xl sm:text-5xl
- 배경 글로우: w-[500px] → w-[600px] sm:w-[800px]
- 글로우 opacity: 0.3 → 0.5
- 로고 드롭 섀도우 추가
- 스케일인 애니메이션 적용
```

### index.css 애니메이션 강화
```
- 로고 펄스 효과 강도 증가
- 스케일인 애니메이션 추가
- 글로우 호흡 효과 추가
```
