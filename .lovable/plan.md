
# 인트로 로딩 페이지 구현 계획

## 개요
사이트 첫 진입 시 동영상이 로딩되는 동안 프리미엄 인트로 화면을 보여주어 사용자 경험을 개선합니다.

---

## 현재 상태 분석

### 존재하는 것
- `PageIntro.tsx` 컴포넌트 (미사용 상태)
- `PageTransitionWrapper` (페이지 간 전환 애니메이션)
- 비디오 로딩 시스템 (`useVideoPlayer` 훅)

### 문제점
- 첫 방문 시 인트로 없이 바로 페이지 렌더링
- 동영상 로딩 중 빈 화면/깜빡임 발생 가능
- `PageIntro.tsx`에 필요한 CSS 애니메이션 미정의

---

## 구현 계획

### 1단계: CSS 애니메이션 추가

`src/index.css`에 인트로 화면용 애니메이션 정의

- `animate-draw-circle`: SVG 원 그리기 애니메이션
- `animate-draw-check`: 체크마크 그리기 애니메이션
- `animate-letter-pop`: 글자 팝업 효과
- `animate-fade-up`: 태그라인 페이드업
- `animate-loading-bar`: 로딩 바 진행 애니메이션

### 2단계: PageIntro 컴포넌트 개선

`src/components/PageIntro.tsx` 수정

- 브랜드 로고 이미지 사용 (현재 SVG 대신)
- 동영상 프리로드 상태와 연동
- 최소 표시 시간 + 동영상 준비 완료 조건 조합
- 부드러운 페이드아웃 전환

### 3단계: App.tsx에 인트로 통합

`src/App.tsx` 수정

- `showIntro` 상태 관리 (첫 방문 시에만)
- `sessionStorage`로 세션 내 재방문 시 스킵
- 인트로 완료 후 메인 콘텐츠 표시

---

## 기술 세부사항

### 로딩 페이지 UX 흐름

```text
┌─────────────────────────────────────────────────────────┐
│                    사이트 첫 진입                        │
└──────────────────────────┬──────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│              PageIntro 컴포넌트 렌더링                   │
│  ┌───────────────────────────────────────────────────┐  │
│  │  • 로고 페이드인 + 펄스 애니메이션                 │  │
│  │  • 브랜드명 글자 순차 팝업                        │  │
│  │  • 태그라인 페이드업                              │  │
│  │  • 하단 로딩 바 진행                              │  │
│  └───────────────────────────────────────────────────┘  │
└──────────────────────────┬──────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│          최소 시간(2초) + 비디오 메타데이터 로드 완료     │
└──────────────────────────┬──────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│             페이드아웃 + 스케일업 전환                   │
└──────────────────────────┬──────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                 메인 홈페이지 표시                       │
│         (비디오가 이미 준비되어 즉시 재생)               │
└─────────────────────────────────────────────────────────┘
```

### 인트로 조건

| 조건 | 동작 |
|------|------|
| 첫 방문 (세션 내) | 인트로 표시 |
| 세션 내 재방문 | 인트로 스킵 |
| 다른 페이지에서 홈 이동 | 인트로 스킵 |

### 수정 파일

| 파일 | 변경 내용 |
|------|----------|
| `src/index.css` | 인트로 애니메이션 keyframes 추가 |
| `src/components/PageIntro.tsx` | 로고 이미지 사용, 동영상 프리로드 연동 |
| `src/App.tsx` | 인트로 상태 관리, 조건부 렌더링 |

---

## 기대 효과

- 동영상 로딩 시간을 브랜딩 경험으로 전환
- 첫인상에서 프리미엄 에이전시 이미지 강화
- 로딩 중 빈 화면/깜빡임 제거
- 부드러운 시작 경험 제공
